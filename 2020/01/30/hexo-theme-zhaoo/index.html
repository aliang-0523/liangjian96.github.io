特性

  • [x] 简约 & 响应式 & 动效
  • [x] highlight 代码高亮
  • [x] Gitalk 评论
  • [x] FancyBox 图片灯箱
  • [x] Pjax 无刷新加载
  • [x] lazyload 图片懒加载
  • [x] LeanCloud 统计访问量
  • [x] 支付宝、微信 打赏
  • [x] 相册页面，图片支持瀑布流加载（justifiedGallery）
  • [x] 格言，可配合插件 hexo-plug-motto 实现动态格言
  • [x] 定制 主题风格、CDN、埋点脚本、社交链接、版权说明 等
